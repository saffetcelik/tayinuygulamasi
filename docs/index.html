Alternatif Kurulum Yöntemleri

Geliştirme veya özelleştirme yapmak istiyorsanız aşağıdaki yöntemleri kullanabilirsiniz

1

Projeyi Klonlama

GitHub'dan projeyi yerel bilgisayarınıza klonlayın:

git clone https://github.com/saffetcelik/tayinuygulamasi.git
cd tayinuygulamasi
2

PostgreSQL Kurulumu ve Yapılandırması

PostgreSQL'i kurun ve veritabanını oluşturun:

# PostgreSQL'e bağlanın
psql -U postgres
# Veritabanı oluşturun
CREATE DATABASE TayinDB;

Not: server/TayinAPI/appsettings.json dosyasında bağlantı dizesini güncelleyin.

3

Backend (.NET) Kurulumu

ASP.NET Core API'yi kurun ve çalıştırın:

cd server/TayinAPI
dotnet restore
dotnet tool install --global dotnet-ef
dotnet ef database update
dotnet run --urls=http://localhost:5000

API şu adreste çalışacak: http://localhost:5000

4

Frontend (React) Kurulumu

React uygulamasını kurun ve çalıştırın:

# Yeni terminal açın
cd client
npm install
npm start

Frontend şu adreste çalışacak: http://localhost:3001

✓

Kurulum Tamamlandı!

Artık uygulamayı kullanmaya başlayabilirsiniz:

👤 Kullanıcı Paneli
http://localhost:3001

(Manuel kurulum için)

👨‍💼 Admin Paneli
http://localhost:3001/admin/panel

(Manuel kurulum için)

⚡ Alternatif: Otomatik Başlatma

Manuel adımlar yerine, proje kök dizininde bulunan PowerShell script'ini kullanabilirsiniz:

# PowerShell'de çalıştırın
./start.ps1

Bu script hem backend hem de frontend'i otomatik olarak başlatır.
