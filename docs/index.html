🚀 Kurulum Adımları

🐳 Docker ile Kurulum (Önerilen)

Önerilen

Tek komutla tüm bileşenleri çalıştırın:

Terminal/PowerShell 
git clone https://github.com/saffetcelik/tayinuygulamasi.git
cd tayinuygulamasi
docker-compose up -d --build
✅ Otomatik PostgreSQL ✅ Migration Uygulama ✅ Demo Veriler ✅ Tek Komut

⚙️ Manuel Kurulum

Gelişmiş
1
Projeyi Klonlama
git clone https://github.com/saffetcelik/tayinuygulamasi.git
cd tayinuygulamasi
2
PostgreSQL Kurulumu

PostgreSQL'i indirin ve kurun. appsettings.json dosyasında bağlantı bilgilerini güncelleyin.

3
Backend Kurulumu
cd server/TayinAPI
dotnet restore
dotnet tool install --global dotnet-ef
dotnet ef database update
4
Frontend Kurulumu
cd ../../client
npm install
5
Uygulamayı Çalıştırma
# Kök dizinde
./start.ps1

# Veya manuel olarak:
# Backend: cd server/TayinAPI && dotnet run --urls=http://localhost:5000
# Frontend: cd client && npm start
